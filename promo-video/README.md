4 条中文静态字幕条（低温手作 / 零添加真食材 / 一码溯源 / 城市合伙人招募），半透明底条，已烧录进成片。

## 音频

- `narration.wav` — 中文旁白合轨（系统「婷婷」TTS，5 段对齐时间轴）
- `n1~5.wav` / `n1~5.aiff` — 各段旁白分轨（中间产物）
- `bgm.wav` — 合成占位背景音乐（C 大调和弦垫底，无版权风险）
- `mixed.wav` — 旁白 + BGM 精修混音（旁白主音，BGM 压低，loudnorm -16 LUFS）

> 正式对外发布建议将 `bgm.wav` 替换为授权招商 BGM，替换后重跑混音即可。

## 制作链路

```
云端 AI 文生视频（4 段 × 5s）
  → 本地 ffmpeg 拼接（含结尾标题卡）
  → PIL 生成字幕条 / 标题卡 PNG
  → 系统 TTS 中文旁白 + 合成 BGM 混音
  → 输出横版（1920×1080）与竖版（1080×1920）
```

## 二次剪辑提示

- 所有分镜参数一致（1080p / 24fps / H.264），可直接用 ffmpeg `concat` 重新拼接。
- 竖版由横版经 `scale + gblur + overlay` 生成，保留完整画面。
- 更换旁白 / BGM 只需重混 `mixed.wav` 后替换音轨（`-c:v copy -c:a aac`）。
