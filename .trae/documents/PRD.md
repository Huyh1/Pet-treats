# 宠物零食售卖网站 - 产品需求文档（PRD）

## 1. 产品概述

「毛茸手作」是一家面向城市宠物主人的高端宠物零食品牌电商网站，主打天然手作、可追溯原料、低敏配方的猫狗零食。
- **核心问题**：解决城市宠物主人对安全、健康、有品质感的宠物零食的需求，让购买体验像在为家人挑选食物一样温暖、有信任感
- **目标用户**：25-45 岁城市中产养宠人群，注重品质、原料透明度与审美趣味，愿意为「手作感」「天然感」支付溢价
- **市场价值**：通过编辑式视觉与故事化叙事，建立品牌差异化，把宠物零食从「日用品」升级为「生活方式选品」

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 访客 | 无需注册 | 浏览商品、查看详情、加入购物车、提交订单 |
| 会员 | 邮箱注册 | 订单历史、收藏夹、地址簿、订阅优惠 |

### 2.2 功能模块

1. **首页**：品牌 Hero、系列导航、畅销精选、品牌故事、原料溯源、用户评价、订阅区块
2. **商品列表页**：分类筛选、排序、商品卡片网格、加入购物车
3. **商品详情页**：商品大图、原料组成、营养分析、喂食建议、用户评价、相关推荐
4. **购物车页**：商品列表、数量调整、小计结算、优惠码
5. **品牌故事页**：创始故事、原料溯源地图、手作工艺流程

## 3. 核心流程

### 3.1 浏览选购流程

用户进入首页 → 浏览畅销精选 / 进入商品列表 → 通过筛选缩小范围 → 点击商品卡片进入详情页 → 查看原料与评价 → 加入购物车 → 进入购物车调整数量 → 应用优惠码 → 进入结算（前端 mock 完成订单）

## 4. 用户界面设计

### 4.1 设计风格

**美学方向：手作编辑式（Editorial Handcraft）**

把宠物零食当作一本独立美食杂志的内容来呈现：温暖、克制、有质感，远离「卡通宠物 + 亮蓝粉嫩」的俗套。

- **主色**：奶油纸色 `#F4EDE0`（背景）+ 深焙可可 `#2B1A12`（文字与主按钮）
- **强调色**：手作橙 `#E2602B`（CTA 与高亮）+ 苔藓绿 `#5C6B3B`（功能型/低敏标签）
- **次色**：羊皮纸 `#E8DCC4`、烤面包金 `#C9A35C`
- **按钮风格**：圆角胶囊型（4px 圆角的微圆角，避免过于卡通），主按钮为深可可底白字，hover 时上浮 2px + 投影
- **字体**：
  - 标题字体：`Fraunces`（带衬线 display，温润手工感）
  - 正文字体：`Noto Sans SC`（中文正文清晰可读）
  - 标签字体：`Space Mono`（小标签、价格、营养数值，带一点工业感对比手工感）
- **布局**：桌面优先，不对称栅格，留白充足，商品卡片采用大图 + 极简信息层级
- **图标 / 装饰**：手绘风插画（骨头、鱼骨、肉块、爪印），SVG 内联，配合照片使用
- **纹理**：纸张噪点叠加、手写体分隔线、邮戳风格的徽章

### 4.3 响应式

- 桌面优先（≥ 1280px 完整布局）
- 平板（768-1279px）：商品网格由 3 列变 2 列，筛选侧栏改为顶部抽屉
- 移动（< 768px）：单列流式，Hero 标题缩至 40px，导航折叠为汉堡菜单，触摸目标 ≥ 44px

### 4.4 3D 场景

不适用。本项目以编辑式 2D 视觉为主，不使用 3D 场景。
